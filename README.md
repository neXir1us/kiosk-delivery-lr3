# Система управления поставками товаров в розничные киоски

Вариант 7 - ЛР2 по дисциплине "Хранилища данных"

## Содержимое репозитория:
- 3NF.sql - SQL код для модели 3NF
- Data_Vault.sql - SQL код для модели Data Vault

## Описание
Система описывает порядок поставок товаров в розничные киоски с использованием двух подходов проектирования БД.

## Сущности системы:
- Города
- Поставщики
- Товары
- Розничные киоски
- Поставки
- Товары в поставках
